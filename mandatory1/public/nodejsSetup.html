Nodejs Setup

How to setup your Node.js express project

Find your favorit IDE to start your next coding project

Create a folder, in that folder make a JavaScript file called app.js

Next we create a package.json file in that file make sure to make {} otherwise it will not be recognize as json file

Next you go to your terminal either in your IDE or open it up on your computer.

Go to your folder where you have your app.js see terminal commands here

Now we have to install a npm package from Node.js open source library

In our json file we put in our dependencies

                {
                      "dependencies":{
                            "express":"^4.17.1",
                            }
                    }

Now run the command "npm install"

This will install the express package, and create a folder node_modules and package-lock.json TODO

We head back to our app.js and import the express package like this 

                const express = require("express");
                    const app = express();

We're calling require which is going to load in the express module we downloaded before thoughe npm. So now this is going to work for any node script. TODO

Visit www.npmjs.com This is going to allow us to take advantage of all of those awesome NPM modules from right inside of our node application. There are an endless amount of awesome useful NPM packages that we can install so we don't have to recreate the wheel from scratch.

Now we are ready to start creating our first crud api application

But before we do that lets install one more npm module called nodemon

Nodemon is a tool that helps develop node.js based applications by automatically restarting the node application when file changes in the directory are detected

We install nodemon through the terminal and install it the recommened way

                npm install -g nodemon

And nodemon will be installed globally to our system path.
