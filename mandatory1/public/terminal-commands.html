Terminal Commands

Basic terminal commands

Command Action
/ (Forward Slash) Top level directory
. (Single Period) Current directory
..(Double Period) Parent directory
~ (Tilde) Home directory
sudo [command] Run command with the security provileges of the super user (super user do)
Change Directory
CREDITS_HERE
cd Home directory
cd [folder] Change directory, e.g. cd Documents
pwd Show your working directory
cd.. Move up to the parent directory
List Directory Contents
CREDITS_HERE
ls Display the name of files and subdirectories in the directory
ls -a List all entries including those with .(period) and ..(double period)
File and Directory Management
CREDITS_HERE
mkdir <dir> Create new folder named <dir>
mkdir <dir1> <dir2> <dir3> Create several folders at once
rmdir <dir> Delete a folder (only workds on empty folders)
rm -R <dir> Delete a folder and its contens


Nodejs terminal commands

Command Action
npm install install your dependencies you have listed in package.json
npm install [package] install that package to your package.json file
npm i [package] install that package to your package.json file
Ctrl + C Kill whatever you're running
